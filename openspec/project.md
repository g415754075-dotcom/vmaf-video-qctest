# VMAF-Video-QCTest 项目背景

## 项目概述

**VMAF-Video-QCTest** 是一款专业的视频质量评估工具，基于 Netflix 开源的 VMAF（Video Multi-Method Assessment Fusion）框架，集成 SSIM、PSNR 等多种客观视频质量评估算法，为影像效果设计师和视频编解码工程师提供精准的视频质量分析服务。

## 项目背景

### 行业痛点

在视频编解码和后期处理工作流程中，工程师需要频繁评估视频质量变化：
- **编码参数调优**：测试不同编码器、码率、分辨率对画质的影响
- **压缩质量验证**：确保压缩后的视频符合质量标准
- **转码效果对比**：对比不同转码方案的画质损失
- **特效渲染检验**：验证视频特效处理后的质量保持度

### 现有问题

1. **工具分散**：VMAF、SSIM、PSNR 等工具各自独立，需要分别运行
2. **操作繁琐**：命令行工具学习成本高，参数配置复杂
3. **结果分析难**：原始数据难以直观理解，缺乏可视化
4. **批量处理弱**：多视频对比时效率低下

### 解决方案

本项目通过 Web 界面整合多种视频质量评估算法，提供：
- 一站式视频质量分析平台
- 直观的可视化结果展示
- 便捷的多视频对比功能
- 详细的质量评估报告导出

## 核心价值

| 价值点 | 描述 |
|--------|------|
| **效率提升** | 减少 80% 的工具切换和命令行操作时间 |
| **决策支持** | 可视化数据帮助快速定位质量问题 |
| **标准化** | 统一的评估流程和报告格式 |
| **协作便利** | Web 端访问便于团队共享结果 |

## 技术选型

- **后端**：Python + FastAPI
- **前端**：React + TypeScript + Tailwind CSS
- **核心库**：Netflix VMAF (libvmaf)
- **视频处理**：FFmpeg
- **数据可视化**：ECharts / Recharts

## 目标用户

1. **影像效果设计师**：需要验证视频特效处理质量
2. **视频编解码工程师**：需要测试和优化编码参数
3. **质量控制人员**：需要批量检验视频交付质量
4. **媒体技术研究者**：需要进行视频质量相关研究

## 项目范围

### MVP 版本（V1.0）

- 单/多视频上传和管理
- VMAF/SSIM/PSNR 质量评估
- 帧级别质量曲线图
- 评估结果对比视图
- 质量报告导出（PDF/JSON）

### 未来规划

- V2.0：批量任务队列、历史记录管理
